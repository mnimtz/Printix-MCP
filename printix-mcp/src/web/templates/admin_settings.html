Entra ID (Azure AD) — Single Sign-On
Ermöglicht die Anmeldung über ein Microsoft-Konto. Benutzer aus jedem Entra-Tenant können sich anmelden — du brauchst nur eine App-Registration in deinem eigenen Tenant.
Einrichtung in 3 Schritten:
  1. Öffne das Azure Portal → App-Registrierungen → Neue Registrierung
  2. Name: Printix MCP, Unterstützte Kontotypen: Konten in einem beliebigen Organisationsverzeichnis (Multi-Tenant)
  3. Umleitungs-URI (Web): {{ entra_redirect_uri }}
Nach dem Erstellen: Client-ID von der Übersichtsseite kopieren, unter Zertifikate & Geheimnisse → Neuer geheimer Clientschlüssel ein Secret erstellen und den Wert kopieren.
Zu finden im Azure Portal → Entra ID → Übersicht. Leer lassen = „common" (alle Tenants).
Wird verschlüsselt gespeichert. Leer lassen = bestehender Wert bleibt erhalten.
